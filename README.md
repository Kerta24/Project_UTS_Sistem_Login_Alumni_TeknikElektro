# Project_UTS_Sistem_Login_Alumni_TeknikElektro
Jurusan Teknik Elektro membutuhkan sebuah sistem informasi yang dapat menyimpan data alumninya. Sistem informasi tersebut nantinya dapat digunakan sebagai sarana penghubung baik antara Jurusan Teknik Elektro dengan alumni maupun sesama alumni. Selain itu para alumni juga dapat memberikan informasi tentang kondisi tempat bekerja sekarang dan informasi lowongan pekerjaan di tempatnya bekerja. Dalam sistem informasi terdapat dua jenis pengguna yaitu alumni dan admin Jurusan Teknik Elektro. Setiap pengguna mempunyai peran dan fungsinya masing-masing. Sebagai alumni, pengguna bisa mendaftarkan diri melalui menu pendaftaran dan membutuhkan persetujuan dari admin untuk memastikan bahwa pengguna adalah alumni mahasiswa Jurusan Teknik Elektro. Di halaman alumni terdapat menu yang dapat digunakan untuk berbagi informasi terkini baik berupa pesan maupun lampiran dokumen yang dapat diakses oleh alumni lain maupun admin. Selain itu juga terdapat menu untuk mengupdate biodata seperti foto profil, informasi pekerjaan, dsb. Fungsi admin dalam sistem informasi ini adalah mengelola seluruh data alumni (lihat, tambah, hapus, edit). Selain itu admin juga mempunyai hak untuk mengontrol informasi yang dibagikan dalam sistem informasi untuk menghindari informasi yang menyimpang dari tujuan pembuatan sistem informasi ini.
